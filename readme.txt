Вводим новую переменную - время до отказа или время между отказами элементов КСЗ



Для построения графиков надежности КСЗ воспользуемся распределением Вейбулла
Тогда нам нужны следующие параметры системы:
- данные о времени до отказа(или время между отказами) КСЗ или входящих в нее СЗИ
- параметры формы(shape), масштаба(scale) и смещения (loc) распределения Вейбулла
===============================================


Пример:

# Генерируем данные
data = weibull_min.rvs(c=2, loc=0, scale=100, size=1000)

c -> shape 
Этот параметр определяет форму распределения Вейбулла. 
Когда c больше 1, распределение имеет форму, близкую к экспоненциальному 
(более надежное в начале, менее надежное со временем). 
В данном случае, c=2 указывает на форму, близкую к распределению Вейбулла.

loc (location): 
Этот параметр задает смещение распределения по оси времени. 
В данном случае, loc=0 означает, что данные не смещены относительно нулевой точки времени.

scale (scale): 
Этот параметр определяет масштаб времени. 
Он влияет на то, насколько быстро или медленно уменьшается функция плотности вероятности (PDF) с увеличением времени. 
В данном случае, scale=100 задает масштаб времени.

size: 
Этот параметр определяет количество случайных значений, которые будут сгенерированы. 
В данном случае, size=1000 указывает на генерацию 1000 случайных значений времени до отказа 
(или времени между отказами) согласно распределению Вейбулла с заданными параметрами.


# Оцениваем параметры распределения Вейбулла по данным
shape, loc, scale = weibull_min.fit(data, loc=0)

# Генерируем данные для построения графика PDF
x = np.linspace(0, max(data), 100)  --> время до отказа
pdf = weibull_min.pdf(x, shape, loc, scale) --> значения функции плотности вероятности для соответствующих значений времени
